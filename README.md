# 使用流程
 * 安裝Ionic
 * 啟用測試=>在Cmd切換到專案資料夾並輸入 ionic serve
 * 包裝成網站 =>在CMD輸入 ionic cordova build --release --verbose
 關於包裝成網站的部分可以參考Ionic 官方文件 => https://ionicframework.com/docs/cli/commands/cordova-build
